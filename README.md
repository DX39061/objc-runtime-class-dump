# [objc-runtime-class-dump](https://github.com/DX39061/objc-runtime-class-dump.git)
## Introduction
dynamic class-dump from objc runtime using frida
## Build
```
npm install
```
## Usage
```
./runner.py --app <app_bundle_id> --module <module_name>
```
## Output
```
class_list.txt
method_list.txt
```
